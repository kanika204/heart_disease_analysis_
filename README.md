Heart Disease Risk Analysis (Tableau + Flask)
This project analyzes heart disease risk factors using interactive data visualizations built in Tableau and presented through a Flask web application.

The goal of the project is to explore relationships between key clinical indicators such as age, cholesterol levels, and blood pressure and their impact on heart disease outcomes.

Project Overview
The analysis focuses on identifying patterns in heart disease data and presenting them through interactive dashboards and a guided data story.

Visualizations were created using Tableau Public and embedded into a Flask-based website for easy access and navigation.

Users can explore:

Interactive dashboards for visual data exploration
A structured data story explaining insights
Summary findings from the dataset
Key Insights
Higher age groups show significantly greater heart disease prevalence
Elevated cholesterol levels correlate with increased risk
Blood pressure patterns vary across age categories
